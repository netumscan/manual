Usar cliente TCP para transferir datos#

Configuración rápida#

   

Nota

Cuando utilice software de terceros para construir un servidor TCP, puede utilizar esta opción para la configuración.

Software Demo#

  • Haga clic en el enlace de abajo para descargar el software demo: Servidor TCP Wi-Fi Netum

Hacer una conexión#

  1. Descargue el software de demostración y descomprimirlo a una carpeta.

  2. Abra el software de demostración, introduzca la información de WiFi y la información del servidor TCP a la que necesita conectarse el escáner y utilice un escáner para escanear el código QR de abajo.

  1. Haga clic para iniciar el servicio o iniciar el servidor TCP a través de la programación.

  2. Espere a que el escáner se conecte activamente al servicio del servidor TCP del software.

Nota

  • Dirección del host: La dirección IP del equipo donde se encuentra el servicio del servidor TCP.

  • Puerto del host: El número de puerto abierto por el servicio TCP Server.
