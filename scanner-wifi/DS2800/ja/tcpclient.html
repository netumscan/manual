TCPクライアントを使用してデータを転送する#

クイック設定#

   

注釈

サードパーティ製のソフトウェアを使用して TCP Server を構築する場合は、このオプションを設定に使用できます。

デモソフトウェア#

  • 以下のリンクをクリックしてデモソフトをダウンロードしてください: Netum Wi-Fi TCP Server

コネクションを作成#

  1. デモソフトウェアをダウンロードし、フォルダに解凍します。

  2. デモソフトウェアを開き、WiFi情報と、スキャナが接続する必要があるTCPサーバー情報を入力します。 スキャナを使って下記のQRコードをスキャンしてください。

  1. クリックしてサービスを開始するか、プログラミングを介して TCP Server を開始します。

  2. スキャナがソフトウェアの TCP Server サービスに積極的に接続するのを待ちます。

注釈

  • ホストアドレス: TCP Server サービスが配置されているコンピュータの IP アドレス。

  • ホスト ポート:TCP Server サービスによって開かれたポート番号。
