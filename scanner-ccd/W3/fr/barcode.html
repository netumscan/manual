EAN-13 vers ISBN#

ISBN est un code de livre, qui donne normalement 13 chiffres. Lorsque la conversion EAN-13 vers ISBN est activée, la sortie est un code ISBN à 10 chiffres.

EAN-13 vers ISSN#

ISSN est un code de magazine, qui donne normalement 13 chiffres. Lorsque la conversion EAN-13 vers ISBN est activée, la sortie est un code ISSN à 10 chiffres.
