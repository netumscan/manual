Définir la plage de longueur#

La longueur d’un code-barres fait référence au nombre de caractères (c’est-à-dire des caractères lisibles par l’homme) contenus dans le code-barres, y compris le chiffre de vérification. La longueur d’un code-barres peut être réglée sur n’importe quelle longueur, dans une plage spécifique.

Note

Lors de la définition des longueurs, les nombres à un seul chiffre doivent toujours être précédés d’un zéro supérieur.

Longueur de portée spécifique#

Permet la lecture des codes-barres avec une longueur spécifique. Scannez d’abord le code-barres avec une longueur dans une plage spécifique, puis scannez le code de réglage à 4 chiffres dans la appendix pour définir les deux longueurs correspondantes L1 et L2. Les L1 et L2 occupent chacun un code de réglage à deux chiffres, et tous les chiffres inférieurs à 2 sont rembourrés avec 0,paramètre : 01-99.

  • Lorsque L1 < L2, L1 est la longueur minimale et L2 est la longueur maximale.

  • Lorsque L1 > L2, L1 est la longueur maximale et L2 est la longueur minimale.

  • Lorsque L1 = L2, seule la longueur fixe (L1/L2) est lue

Astuce

  • Seuls 4-8 caractères sont autorisés à être lus, alors scannez le code de réglage numérique « 0 » « 4 » « 0 » « 0 » « 8 » ou « 0 » « 8 » « 0 » « 4 »

  • Pour ne lire que 12 caractères, scannez le code de réglage numérique « 1 » « 2 » « 1 » « 2 » « 2 »

N’importe quelle longueur#

Pour lire les codes-barres de n’importe quelle longueur, scannez n’importe quelle longueur sans scanner le code de réglage numérique. Par défaut: 0-99

Définir la plage de longueur#

La longueur d’un code-barres fait référence au nombre de caractères (c’est-à-dire des caractères lisibles par l’homme) contenus dans le code-barres, y compris le chiffre de vérification. La longueur d’un code-barres peut être réglée sur n’importe quelle longueur, dans une plage spécifique.

Note

Lors de la définition des longueurs, les nombres à un seul chiffre doivent toujours être précédés d’un zéro supérieur.

Longueur de portée spécifique#

Permet la lecture des codes-barres avec une longueur spécifique. Scannez d’abord le code-barres avec une longueur dans une plage spécifique, puis scannez le code de réglage à 4 chiffres dans la appendix pour définir les deux longueurs correspondantes L1 et L2. Les L1 et L2 occupent chacun un code de réglage à deux chiffres, et tous les chiffres inférieurs à 2 sont rembourrés avec 0,paramètre : 01-99.

  • Lorsque L1 < L2, L1 est la longueur minimale et L2 est la longueur maximale.

  • Lorsque L1 > L2, L1 est la longueur maximale et L2 est la longueur minimale.

  • Lorsque L1 = L2, seule la longueur fixe (L1/L2) est lue

Astuce

  • Seuls 4-8 caractères sont autorisés à être lus, alors scannez le code de réglage numérique « 0 » « 4 » « 0 » « 0 » « 8 » ou « 0 » « 8 » « 0 » « 4 »

  • Pour ne lire que 12 caractères, scannez le code de réglage numérique « 1 » « 2 » « 1 » « 2 » « 2 »

N’importe quelle longueur#

Pour lire les codes-barres de n’importe quelle longueur, scannez n’importe quelle longueur sans scanner le code de réglage numérique. Par défaut: 0-99

Formats de début et de fin de caractères#

Note

  • ABCD/ABCD Les caractères de début et de fin peuvent être l’un des quatre caractères suivants : « A », « B », « C », « D »;

  • ABCD/TN*E Le caractère de départ peut être l’un des quatre caractères suivants : « A », « B », « C », « D »; le caractère final peut être l’un des quatre caractères suivants : « T », « N », « * », « E ».
