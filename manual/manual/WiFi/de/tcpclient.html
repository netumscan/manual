Benutze TCP Client um Daten zu übertragen#

Schnellkonfiguration#

   

Bemerkung

Wenn Sie Software von Drittanbietern zum Erstellen eines TCP-Servers verwenden, können Sie diese Option für die Konfiguration verwenden.

Demo-Software#

  • Klicken Sie auf den untenstehenden Link, um die Demo-Software herunterzuladen: Netum Wi-Fi TCP Server

Verbindung herstellen#

  1. Laden Sie die Demo-Software herunter und entpacken Sie sie in einen Ordner.

  2. Öffnen Sie die Demo-Software, geben Sie die WiFi-Informationen und TCP-Server-Informationen ein, mit denen der Scanner sich verbinden muss und verwenden Sie einen Scanner, um den QR-Code unten zu scannen.

  1. Klicken Sie zum Starten des Dienstes oder starten Sie TCP Server durch Programmierung.

  2. Warten Sie, bis der Scanner sich aktiv mit dem TCP Server Service der Software verbindet.

Bemerkung

  • Hostadresse: Die IP-Adresse des Computers, auf dem sich der TCP Server Dienst befindet.

  • Host-Port: Die vom TCP Server Dienst geöffnete Portnummer.
