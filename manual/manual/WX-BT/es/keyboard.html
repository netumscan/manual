Idioma del teclado#

  • Si está utilizando un teclado francés, escanee el código de barras de comandos French.

  • Si utiliza un teclado de EEUU, puede ignorar este paso (el teclado estadounidense se utiliza por defecto en China).

  • Si los datos escaneados y transmitidos están desgranados, puede ser que el idioma del método de entrada del ordenador no corresponda con el idioma del escáner.

[1]

TH configuración de idioma, el módulo de escaneo también necesita una salida TH.

[2]

El teclado universal internacional ALT sólo es aplicable a sistemas Windows y no se ve afectado por los ajustes del teclado del equipo, pero afectará a la velocidad de transmisión.
